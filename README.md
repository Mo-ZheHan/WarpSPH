<img height="300" src="./sph_logo.png" width="300" alt="Logo"/>

## 项目描述

该项目实现了SPH模拟。

## 配置方法

1. 安装 poetry 并执行 `poetry install`
2. 执行`source $(poetry env info -p)/bin/activate`激活环境
